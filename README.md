# Task Management API

## Features
- User Registration
- User Login
- JWT Authentication
- Create Task
- View Tasks
- Update Task
- Delete Task
- Filter Tasks by Status

## Technologies Used
- Flask
- Flask SQLAlchemy
- SQLite
- Flask-JWT-Extended
- Werkzeug

## Run the Project

pip install -r requirements.txt

python app.py

Server runs on:
http://127.0.0.1:5000
